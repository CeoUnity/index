Corgi Steps

Creating a level

  1. Add a camera with a camera controller

    Prefabs > Camera > Regular camera

  2. Ui Camera with input manager

  3. Add a game manager

  4. Add a LevelStart

    Prefabs > LevelManagers > LevelStart
    1. Create a new game object and rename it LevelManager
    2. zero positions of those and make LevelStart child of LevelManager
    3. on LevelManager increase level bounds

Importing an image asset

  1. Filter mode: point no filter

  2. Compresion: None

  3. Pixels Per Unity

Creating the avatar

  1. adjust the sprites with sprite editor

  2. drag those pngs to scene to start creating the animations

  3. create a prefab

  4. Add a characther component to the player prefab

  5. Change Character type to player on Character component

  6. Change Player Id on Character component to Player1

  7. Click on the AutoBuild Player Character button on Character component

  8. Edit Player Character scripts on Character component

Adding Player to the level spawn

  1. On LevelManager choose the player on Player Prefabs component

Create A Start Menu

  1. create a new scene

  2. create a new canvas as child of the scene with an img as a background

    • Canvas Component:
      1. Canvas Scaler
        • UI scale mode: Scale with screen size
        • Resolution:(4k) 3840 x 2160
        • Resolution:(hd) 1920 x 1080
        • the anchor points: hold( alt ) and select last option
        • the Texture Type of the img must be set as Sprite
    • Create menu buttons
      • create all buttons
      • create a new empty GO N rename it MainMenu
        1. set the anchor points to scale to all directions
        2. zero positions in Rect Transform
        3. group all btns inside of it
      • Options Menu
        1. create a new empty GO N rename it Options
        2. set width: 1000 N height: 700
        3. add a img as a child of Options
        4. set anchors and zero positions
        5. set the src img
        6. add a text as a child of Options
        7. create a new Slider as a child of Options
    • Creating the script for menu
      • create a new empty GO N rename it MenuManager
      • create a new script N make it child of MenuManager 
      • using UnityEngine.SceneManagement;

[SerializeField] private string carregarLevel;
[SerializeField] private GameObject menuInicial;
[SerializeField] private GameObject opcoes;

public void Jogar() {
    SceneManagemer.LoadScene( carregarLevel );
}

public void AbrirOpcoes() {
    menuInicial.SetActive( false );
    opcoes.SetActive( true );
}

public void FecharOpcoes() {
    menuInicial.SetActive( true );
    opcoes.SetActive( false );
}

public void SairDoJogo() {
    Debug.Log( "Application.Quit" );
    Application.Quit();
}
